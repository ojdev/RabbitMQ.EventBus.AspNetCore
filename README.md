该包为一个基于官方RabbitMQ.Client的二次封装包，专门针对Asp.Net Core项目进行开发，在微服务中进行消息的传递使用起来比较方便。

目前功能：

- [x] 发布/订阅
- [x] 死信队列
- [x] RPC功能（实验性）
